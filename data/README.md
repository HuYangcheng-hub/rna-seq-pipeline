# Data Directory

Place FASTQ files here using the naming convention: `{sample}_R1.fastq.gz` / `{sample}_R2.fastq.gz`.

将 FASTQ 文件放在此目录，命名格式：`{sample}_R1.fastq.gz` / `{sample}_R2.fastq.gz`

## Quick Test Data / 测试数据

```bash
bash scripts/download_test_data.sh
```

Large FASTQ files are ignored by git. 大型 FASTQ 文件已被 git 忽略。
