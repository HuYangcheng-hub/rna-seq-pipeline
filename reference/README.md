# Reference Directory

Put genome FASTA, GTF/GFF annotation files, and STAR indexes here.

The default `config/config.yaml` points to a small chr22 demo reference path:

- `reference/chr22/Homo_sapiens.GRCh38.dna.chromosome.22.fa`
- `reference/chr22/chr22.gtf`
- `reference/chr22/star_index/`

Large reference files and generated STAR indexes are intentionally ignored by git.
